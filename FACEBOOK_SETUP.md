# Настройка авторизации через Facebook

## Инструкция по настройке

1. Перейдите на https://developers.facebook.com/
2. Создайте новое приложение или выберите существующее
3. Получите **App ID** из настроек приложения
4. В настройках приложения добавьте:
   - **Valid OAuth Redirect URIs**: `http://localhost:5173` (для разработки) и ваш домен (для продакшена)
   - **App Domains**: ваш домен
5. Замените `YOUR_APP_ID` в файле `index.html` на ваш реальный App ID

## Использование переменных окружения (рекомендуется)

1. Создайте файл `.env` в корне проекта
2. Добавьте строку:
   ```
   REACT_APP_FACEBOOK_APP_ID=ваш_app_id
   ```
3. В `index.html` замените `'YOUR_APP_ID'` на `process.env.REACT_APP_FACEBOOK_APP_ID || 'YOUR_APP_ID'`

## Примечание

Если App ID не настроен, при клике на кнопку Facebook будет открываться страница входа Facebook (демо-режим).
