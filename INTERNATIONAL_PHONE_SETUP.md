## 📝 Как это работает:

1. **Пользователь выбирает страну** из выпадающего списка
2. **Вводит номер телефона** - автоматически форматируется
3. **Номер отправляется в международном формате** (код страны + номер)
4. **Green API получает номер** в правильном формате для WhatsApp

## 💡 Примеры:

### Беларусь:
- Выбираете: 🇧🇾 Беларусь (+375)
- Вводите: `29 123 45 67`
- Результат: `375291234567`

### Россия:
- Выбираете: 🇷🇺 Россия/Казахстан (+7)
- Вводите: `999 123 45 67`
- Результат: `79991234567`

### США:
- Выбираете: 🇺🇸 США/Канада (+1)
- Вводите: `555 123 4567`
- Результат: `15551234567`

## ⚠️ Важно:

- Номер должен быть в формате: **код страны + номер** (без +)
- Минимальная длина: 10 цифр
- Максимальная длина: 15 цифр (стандарт E.164)
- Код страны выбирается автоматически при выборе страны

## 🔧 Если нужно добавить больше стран:

Откройте `src/components/PhoneInput.jsx` и добавьте в массив `COUNTRY_CODES`:

```javascript
{ code: 'XX', name: 'Название страны', flag: '🇺🇳' }
```

Где `XX` - код страны, `flag` - эмодзи флага.
