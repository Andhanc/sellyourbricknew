## 🔄 Анимации

### Появление бейджа:
```
Время: 0.3s
Эффект: Плавное появление сверху вниз
```

### Пульсация иконки часов:
```
Время: 2s (бесконечно)
Эффект: Opacity 1 → 0.5 → 1
```

### Обновление таймера:
```
Частота: Каждую 1 секунду
Формат: "71ч 45м" → "71ч 44м" → ...
```

## 📱 Мобильная версия

### Бейдж на карточке:
- Размер уменьшается пропорционально
- Остается в левом верхнем углу
- Текст остается читаемым

### Блок на странице объекта:
- Переключается на вертикальную компоновку
- Иконка центрируется
- Текст выравнивается по центру
- Все элементы масштабируются

## 🎯 Сценарии использования

### Сценарий 1: Успешная покупка

```
1. Пользователь А отправляет запрос
   → Объект бронируется
   → Появляется зеленый блок с таймером
   
2. Админ видит запрос в админке
   → Связывается с пользователем
   
3. Сделка завершается
   → Админ нажимает "Завершить"
   → Бронь снимается
   → Объект снова доступен
```

### Сценарий 2: Отмена запроса

```
1. Пользователь А отправляет запрос
   → Объект бронируется
   
2. Пользователь Б пытается купить тот же объект
   → Видит оранжевый блок "Объект забронирован"
   → Кнопка заблокирована
   
3. Пользователь А передумал
   → Админ нажимает "Отменить"
   → Бронь снимается
   
4. Пользователь Б обновляет страницу
   → Блок исчезает
   → Кнопка становится активной
   → Может отправить запрос
```

### Сценарий 3: Истечение времени

```
1. Пользователь А отправляет запрос
   → Объект бронируется на 72 часа
   → Таймер: "71ч 59м"
   
2. Проходит 72 часа
   → Таймер: "0ч 0м"
   → Бронь автоматически снимается
   → Блок исчезает
   
3. Объект снова доступен для всех
```

## 🎨 Цветовая палитра

### Зеленый (своя бронь):
- **Primary**: `#0ABAB5`
- **Secondary**: `#089a95`
- **Использование**: Успех, подтверждение, своя бронь

### Оранжевый (чужая бронь):
- **Primary**: `#ffa726`
- **Secondary**: `#fb8c00`
- **Использование**: Предупреждение, блокировка, чужая бронь

### Серый (заблокировано):
- **Opacity**: `0.5`
- **Cursor**: `not-allowed`
- **Использование**: Неактивные элементы

## ✨ Дополнительные детали

### Иконки:
- 🔒 `FiLock` - Замок (чужая бронь, блокировка)
- 🕐 `FiClock` - Часы (таймер, своя бронь)

### Шрифты:
- **Заголовок**: 1.125rem, weight: 600
- **Таймер**: 1rem, weight: 500
- **Сообщение**: 0.875rem, weight: 400

- **Блок**: padding: 1.5rem
- **Бейдж**: padding: 8px 14px
- **Иконка**: 48x48px (блок), 24x24px (бейдж)
